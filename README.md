# アプリの説明
このアプリはプログラミング学習関連の投票アプリです。<br>
オススメオンライン教材、MacBookのスペック、オススメTwitterユーザーの3項目を投票し、閲覧する事ができます。<br><br><br>

# UPDATE予定
・画像の投稿機能と、アマゾンや楽天へのリンクを付与したオススメ書籍の投票項目を追加<br>
・学習ロードマップの投票項目を追加<br><br><br>
